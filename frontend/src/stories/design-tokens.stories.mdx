# Design Tokens

Authoritative source: `src/styles/tokens.css` and `tailwind.config.ts`.

## Color

- `--color-surface-*`: `surface-base`, `surface-raised`, `surface-overlay`, `surface-sunken`, `surface-inverse`, `surface-brand-subtle`
- `--color-text-*`: `text-primary`, `text-secondary`, `text-muted`, `text-disabled`, `text-on-brand`, `text-on-inverse`, `text-link`, `text-danger`, `text-on-accent`
- `--color-border-*`: `border-subtle`, `border-default`, `border-strong`, `border-focus`
- `--color-brand-*`: `brand-50` ... `brand-900`
- `--color-{success,warning,danger,info}-*`: `subtle`, `muted`, `solid`, `solid-hover`, `on-solid`, `border`
- `--color-dv-*`: `dv-1` ... `dv-10`, `dv-seq-1` ... `dv-seq-9`, `dv-div-cold`, `dv-div-mid`, `dv-div-hot`
